### **Responsive Landing Page**

**PQ:** How can we construct responsive websites with advanced CSS effects?

**Overview:** You have been hired to create a responsive [landing page](https://unbounce.com/landing-page-articles/what-is-a-landing-page/#:~:text=In%20digital%20marketing%2C%20a%20landing,or%20registration\)%20will%20take%20place.) for one of the following companies of your choice:

| Option 1: StarVoyage [(Branding Guide)](https://docs.google.com/document/u/0/d/1qfVd8tDGWj9dCDIMAEhSGsLx36KPQi_y3kILewgSQRk/edit) Premium space tourism company offering orbital flights and lunar excursions. Target audience: Luxury adventure travelers Key features: Flight booking, safety records, training programs | Option 2: SkyCity Transport [(Branding Guide)](https://docs.google.com/document/u/0/d/1qfVd8tDGWj9dCDIMAEhSGsLx36KPQi_y3kILewgSQRk/edit) Urban air taxi service offering flying car rentals and scheduled air transport. Target audience: Urban professionals Key features: Route mapping, instant booking, safety features  | Option 3: HyperLoop Express [(Branding Guide)](https://docs.google.com/document/u/0/d/1qfVd8tDGWj9dCDIMAEhSGsLx36KPQi_y3kILewgSQRk/edit) Ultra-fast vacuum tube transport between major cities. Target audience: Business travelers and commuters Key features: Route booking, membership programs, real-time tracking  | Option 4: DeepSea Adventures [(Branding Guide)](https://docs.google.com/document/u/0/d/1qfVd8tDGWj9dCDIMAEhSGsLx36KPQi_y3kILewgSQRk/edit) Luxury submarine tours to underwater cities and marine habitats. Target audience: Exotic travel enthusiasts Key features: Tour booking, marine life guides, safety protocols  |

**Requirements:** Your landing page must include the following **mandatory** elements that are required by the client.

* The name of the company with a brief, one sentence description underneath  
* An input and a button that allows the user to be added to the subscription list. (You donâ€™t have to configure the subscription list code)  
* Responsive website that displays properly on mobile, tablet, and desktop devices  
* Includes the following items:  
  * SVG Icons  
  * CSS Transforms

* Adherence, as much as possible, to the branding guidelines for the company you choose

### **Rubric for Responsive Landing Page Project (30 Points)**

| Category | Exemplary (5 points) | Solid (4 points) | Developing (3 points) | Needs Attention (2 points) | No Credit (0 points) |
| ----- | ----- | ----- | ----- | ----- | ----- |
| **Responsive Design** | Fully responsive; displays perfectly on mobile, tablet, and desktop; seamless layout adjustments; effective use of media queries. | Responsive; displays well on all devices with minor layout flaws; effective breakpoints, but transitions could be smoother. | Partially responsive; noticeable layout issues on one or more device sizes; media queries need improvement. | Lacks adequate responsiveness; significant issues on multiple device sizes; incorrect or absent media queries. | Not responsive. |
| **HTML Structure** | Consistent and appropriate use of semantic HTML tags; logical structure facilitates accessibility and readability. | Semantic tags used in most parts; logical structure but minor accessibility improvements needed. | Some semantic tags used; inconsistent or inappropriate usage in key areas; content structure is somewhat logical. | Rarely uses semantic tags; content structure lacks logic, making navigation difficult. | No semantic tags used. |
| **CSS (Animations, Transforms, Filters, Effects)** | Creative and effective use of animations, transforms, and filters; effects enhance appearance, are seamless, and improve user engagement. | Effective use of animations, transforms, and filters; minor improvements could make effects more polished or creative. | Basic or inconsistent use of animations, transforms, and filters. | Poorly executed effects; noticeable errors or minimal usage. | No CSS animations, transforms, or filters used. |
| **Visual Creativity and Aesthetics** | Visually stunning; thoughtful use of branding guidelines, colors, typography, and layout; cohesive and exceptional user experience. | Visually appealing; adheres to branding guidelines; minor inconsistencies or missed opportunities for creativity. | Follows branding guidelines but lacks creativity; inconsistent design elements. | Disconnected design; significant issues in color, typography, or adherence to branding. | No visual appeal; does not follow branding guidelines. |
